# Flappy-Bird
A Simple Flappy Bird Game with new features like Focus mode and adjustable themes and skins.
this is the Source code with complete assets.

# Controls
Space - Flap
Enter - Selection in menu
Up, Down, Left, Right - Menu Navigation


# Gameplay and Settings
Game HomeScreen Contains 2 Different gameplay modes - 
1. Free play - Classic Endless Flappy Bird gameplay
2. Focus Mode Classic gameplay features with focus and reaction time estimations based on the user inputs.

3. Settings - The game UI can be modified in the Settings Bird Skins, Background and Pipes can Be customised on various themes
   themes can be added in the Game from the file 'Flappy_Bird.py' and assets folder contains all the gameplay assets

# Requirements for running the game code
 To rum the Code primary requirements are - Python 3.13, SqlAlchemy, Postgresql, python Pygame library.
